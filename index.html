Comment c'est calculé

Généré à chaque synchronisation à partir de tes données : duel sur course de référence, volume et cumul à date égale, forme CTL/TSB, ACWR croisé avec ton bouclier de renforcement, répartition cardio, signature de cadence, habitudes, usure du matériel. Aucune donnée n'est envoyée à un serveur — tout est calculé dans le navigateur.

⚙️ Application API Strava

Crée ton application sur strava.com/settings/api (« Authorization Callback Domain » = le domaine de cette page, par ex. cosmin332.github.io), puis recopie les identifiants. Stockés sur cet appareil (ou dans config.js du dépôt pour tous les appareils).
